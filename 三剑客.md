- 查看版本：git --version
### config 的三个作用域
```
git config --local 只对某个仓库有效。缺省等同于local
git config --global 对当前用户所有仓库有效（常用）
git config --system 对系统所有登录的用户有效（不常用）
```
### 显示config的配置，加 --lsit
```
git config --lsit 显示所有作用域的配置
git config --lsit --local
git config --list --global
git config --list --system
```
